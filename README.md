AWS VPC, RDS & MySQL Lab
This lab creates a secure AWS network, sets up a managed MySQL database, connects from a Linux server, and runs SQL tasks including creating tables, inserting data, and joining records.

What it does
Builds a custom VPC with public and private subnets
Deploys an RDS MySQL instance following all lab rules
Sets security rules so only the Linux server can access the database
Connects via SSH to the Linux server and installs the MySQL client
Creates two tables: RESTART and CLOUD_PRACTITIONER
Inserts sample data and runs an inner join to show combined results
Provides all required steps for submission and cleanup
